# oxcovid19-dashboard
A syndemic COVID-19 dashboard based on the [OxCOVID-19 database](https://covid19.eng.ox.ac.uk/).

## Installation

Change directory to the repository and run a server, e.g.:
```
python3 -m http.server 1234
```
